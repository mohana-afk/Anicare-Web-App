# Anicare – Animal Care Web Application

Anicare is a web-based application designed to provide basic guidance for animal care, especially in rural areas with limited access to veterinary support.

## Live Demo
https://ani-care-rural-4b943050.base44.app

## Features
- Animal care guidance
- Simple and accessible interface
- User input-based suggestions (prototype)

## How It Works
1. User selects animal type
2. Inputs their concern
3. System provides basic guidance

## Tech Used
- Web-based development platform
- AI-assisted development tools

## Status
Prototype (UI implemented, logic in progress)
